
第14章  用户管理
===============================

Proxmox VE支持多种身份认证方式，例如Linux PAM，内部Proxmox VE认证服务，微软活动目录。

基于角色的用户和权限管理覆盖了所有对象（虚拟机，存储，节点等），能够实现细粒度的访问控制。
